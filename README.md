# Toy Store

## Overview

This project contains a toy store program with a GUI built with Java, JavaFX, and Maven.

## Features

- Search all toys in the database, or serach by name
- Add a new toy into with a category (e.g., figure, puzzle, video game)
- Remove a toy from the dtabase using its serial number

## Getting Started

### Prerequisites

- [Java SDK 21](https://www.oracle.com/ca-en/java/technologies/downloads/#jdk21)
- [JavaFX](https://openjfx.io/openjfx-docs/)

### Installation

1. Clone the repo on your local machine:

```bash
git clone https://github.com/nalhe627/toy-store.git
```

2. Move into the project's root directory:

```bash
cd toy-store
```

### Build & Run the Program

> [!WARNING]
> The project was done done using [VSCode](https://code.visualstudio.com/) and the [Java Extension Pack](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack), and therefore is able to run in VSCode. I currently haven't figured out how to run the project without it.
